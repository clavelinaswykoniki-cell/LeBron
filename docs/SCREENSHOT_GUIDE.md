# 截图指南

> 给「作品集 / README / 朋友圈分享」准备的截图清单。
> 用**微信开发者工具**截，放 `docs/screenshots/` 目录，README 引用相对路径。

---

## 设备建议

- **模拟器**：iPhone 14 Pro（390×844）或 iPhone SE（375×667）任选
- **真机**：iPhone 12+ / 现代安卓机
- **暗色背景**已经是项目主题色（紫金 + 深紫），不用切系统主题

---

## 8 张关键截图

### 1. 首页 hero + 命中卡（`01-home.png`）

**操作**：
1. 打开小程序 → onLoad 自动搜 "8 分释兵权"
2. 滚动到 hero 区域，确保 jersey #23 + 紫金 LOGO + 数据徽章可见
3. **关键元素**：hero-stats（150 / 441 / 38）+ jersey-card "23"

**用途**：README 顶图、朋友圈第一张、作品集封面

---

### 2. 段位浮条 + 关于页（`02-about.png`）

**操作**：
1. 至少点 11 张不同卡（触发青铜→白银）
2. 复制几次（触发 first_copy 勋章）
3. 截首页 hero 下方的"段位浮条"（金紫渐变 strip）
4. 点段位浮条进入 `/pages/about/about` 关于页
5. 截关于页：段位卡 + 勋章墙 + 数据看板

**关键元素**：
- 当前段位（如"白银詹蜜"+ 进度条）
- 勋章墙（解锁高亮 / 未解锁灰显）
- 项目数据（150 / 441 / 38）

**用途**：展示游戏化设计 + 数据持久化

---

### 3. 23 号秘藏彩蛋（`03-easter.png`）

**操作**：
1. 首页 → 长按 jersey-card（23 号区域）约 800ms
2. 进入 `/pages/easter/easter`
3. 截整页：巨型 "23" 水印 + "Strive for Greatness" 大字 + 7 个里程碑卡片

**关键元素**：
- 600rpx 半透明数字水印
- 紫金主题
- 里程碑列表（4 冠 4 FMVP / 历史得分王 / ...）

**用途**：展示彩蛋设计、紫金视觉、动效细节

---

### 4. 球迷测试 - 答题中（`04-quiz-question.png`）

**操作**：
1. 首页 → 菜单 → "🧠 球迷测试"
2. 进入 `/pages/quiz/quiz`
3. 截第 2 或第 3 题（进度条已显示），3 个选项可见

**关键元素**：
- 进度条 "第 2 / 5 题"
- 题面（如 "2018 年总决赛被勇士横扫，你怎么看？"）
- 3 个选项按钮（蜜 / 中立 / 黑 倾向）

**用途**：展示交互设计、中性化打分

---

### 5. 球迷测试 - 结果页（`05-quiz-result.png`）

**操作**：
1. 接 #4，全部按"中立" → 答完 5 题
2. 显示 "中立观察者" 结果页
3. 或者全选"蜜"选项 → 显示 "王牌詹蜜"
4. 截结果页：大字称号 + 描述 + 3 个按钮（复制 / 重答 / 回首页）

**关键元素**：
- 64rpx 金色称号
- 中性化描述文案
- 操作按钮

**用途**：展示完整闭环

---

### 6. 复制爽感（`06-copy-feedback.png`）

**操作**：
1. 首页搜 "8 分" → 命中卡
2. 点 "一键复制" 按钮
3. **快速**截图捕捉金光特效 + Toast 那一瞬
4. 如果截不到金光（700ms 太快），至少截 Toast "已复制"

**关键元素**：
- Toast "已复制" + ✓ 图标
- （理想）金光残影

**用途**：展示反馈细节、UX 关注度

---

### 7. 隐私页（`07-privacy.png`）

**操作**：
1. 首页底部 → 菜单"🛡️ 隐私说明" 或 footer "隐私政策"
2. 进入 `/pages/privacy/privacy`
3. 截整页 5 段政策

**关键元素**：
- 标题 "隐私政策"
- 5 段：不收集 / 本地存储 / 复制说明 / 内容立场 / 联系方式
- 紫底白字

**用途**：展示合规态度、微信审核就绪

---

### 8. 错误处理 / 空状态（`08-empty-state.png`）

**操作**：
1. 首页 → 清空 textarea → 点"生成反驳"
2. Toast "先输入一个黑点（试试 8分 / Excel球王）" 出现
3. 卡片区显示 empty-state "这枪还没装弹"
4. 截 Toast + empty-state 同时可见的一瞬

**关键元素**：
- 友好 Toast 提示
- empty-state 文案（有娱乐感）

**用途**：展示边界 UX、错误兜底意识

---

## 加分截图（可选）

### 长截图 - 完整结果卡（`bonus-full-card.png`）

搜 "Excel 球王" → 截一整张展开后的结果卡（含 facts / tags / 长拆 / 封口 / 口播）。
**用途**：展示信息密度、结构化反驳。

### 分类筛选（`bonus-category-filter.png`）

横向滚动 category-chip 区，截选中"2011 总决赛"或"抱团换队论"时的卡片列表。
**用途**：展示分类能力。

### 微动效 GIF（`bonus-animation.gif`）

用 QuickTime / 微信开发者工具录屏，时长 5-10s，覆盖：
- 长按 jersey → 跳彩蛋
- 复制按钮 → 金光特效
- 段位升级 → Toast

**用途**：作品集首页 hero GIF，朋友圈短视频。
