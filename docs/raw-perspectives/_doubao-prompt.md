# 任务

帮我从**抖音为主**（B 站 / 小红书 / 虎扑 / 知乎为辅）搜罗 NBA 詹姆斯 7 个核心黑点的中文网络观点，给「LeBron 詹黑逻辑拆解器」反驳卡建素材库。

# 搜罗目标

每个黑点都要覆盖 5 个维度：

1. **詹黑博主原话**（不同博主、不同切入角度）
2. **詹蜜博主反击话术**（不同博主、不同反击策略 —— 这一节最重要）
3. **抖音评论区典型对线**（高赞黑 vs 高赞蜜的正面交锋）
4. **关键数据 / 事实点**（可被核查的数字）
5. **高信息量视频链接**（讲事实/讲战术/讲历史的，不是单纯吹/黑）

# 严格要求

1. **抖音为主**：至少 60% 的引用要来自抖音
2. **同一个黑点要展示多个博主的不同切入角度** —— 不要 5 条都是同一类观点
3. **詹蜜反击话术** 这一节是重点 —— 我要看大家用什么逻辑/数据/比喻去反击，**这一节字数应该和詹黑那节差不多甚至更多**
4. **引用原话，不要改写、不要浓缩**
5. **标注来源博主 + 影响力**（点赞 / 播放量级，没有就写「未知」）
6. **不要主观加观点**、不要总结、不要给建议 —— 纯搜罗
7. 中文输出
8. 某节搜不到就写「未搜到」，但不要跳过整个黑点
9. **7 个黑点全跑完**，哪怕每个稍微短一点也要 7 个都覆盖
10. 链接搜不到不强求，但博主名一定要

# 提示

- 抖音搜「詹姆斯 [关键词]」会有大量内容
